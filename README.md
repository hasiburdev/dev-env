# Development Environment Configuration Files

## How to setup

1. Clone the repository

With HTTPS
```bash
git clone https://github.com/hasiburdev/dev-env.git
```

With Git
```bash
git clone git@github.com:hasiburdev/dev-env.git
```

2. Give executable permission to the setup script
```bash
chmod -R +x ./dev-env-setup
```

3. Run the `dev-env-setup` script
```bash
./dev-env-setup
```
This script will setup the necessary shell script permissions and additional configurations.
